# Expense-Tracker

💰 Expense Tracker

A simple and responsive Expense Tracker Web Application that helps users manage their daily income and expenses efficiently.
Track transactions, monitor balance, and visualize spending patterns in one place.

🚀 Live Demo

🔗 View Live Project

📌 Features

➕ Add Income & Expense Transactions

🗑 Delete Transactions

💵 Real-time Balance Calculation

📊 Income & Expense Summary

💾 Local Storage Support (Data persists after refresh)

📱 Responsive Design

🛠 Tech Stack

Frontend: HTML5, CSS3, JavaScript

Deployment: GitHub Pages / Netlify

⚙️ Installation & Setup

1️⃣ Clone the repository

git clone https://github.com/your-username/expense-tracker.git

2️⃣ Navigate to the project folder

cd expense-tracker

3️⃣ Open index.html in your browser
OR use Live Server in VS Code.
